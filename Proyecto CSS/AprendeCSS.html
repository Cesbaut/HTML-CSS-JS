MODELADO DE CAJAS



Espacios en las cajas


El modelo de cajas de CSS es un concepto fundamental que describe cómo se representan y se estructuran los elementos HTML en una página web mediante CSS. Cada elemento HTML se representa como una caja rectangular que consta de cuatro áreas principales: contenido, relleno (padding), borde (border) y margen (margin). Estas áreas son esenciales para el posicionamiento y la apariencia de los elementos en la página.

contenido

Es el área donde se muestra el contenido real del elemento, como texto, imágenes, vídeos u otros elementos HTML. Su tamaño y dimensiones pueden ajustarse utilizando propiedades como width y height.


padding

Es el espacio entre el contenido del elemento y su borde. Se utiliza para separar el contenido del borde, brindando un espacio interno. Puedes controlar el relleno utilizando las propiedades padding-top, padding-right, padding-bottom y padding-left.


border

Es la linea de borde que puede contener tu caja. Esta tiene tres caracteristicas, el primero es un tamaño, el segundo el tipo de borde que aplicara y el tercero el color. Se define mediante la propiedad border.


margin

Es el espacio que rodea el elemento y separa este elemento de otros elementos circundantes. Los márgenes no tienen color ni borde, solo representan espacios en blanco entre elementos. Puedes controlar los márgenes utilizando las propiedades margin-top, margin-right, margin-bottom y margin-left.


Recordando otros comandos.

  • background-color se utiliza para cambiar el color de fondo.
  • border-radius se usa para poner las esquinas redondeadas.
  • width es para poner un ancho a la caja
  • height: para el alto
                
                .contenido{
                    background-color: blue;
                    padding: 5px 10px 12px 20px;
                    border: 1px solid green;
                    border-radius: 4px;
                    margin: 15px;
                    width: 50px;
                    height: 50px;
                    display: inline-block;
                }

Declaraciones y concatenaciones

En lugar de escribir en tu codigo de css los valores para cada lado de un atributo, por ejemplo:

            
            .contenido{
                margin-top: 5px;
                margin-left: 6px;
                margin-bottom: 5px;
                margin-right: 6px;
            }

Es mejor concatenarlo en uno, esto se puede aplicar a los tres espacios de cajas que vimos anteriormente.

                
                .contenido{
                    margin: 5px 6px 5px 6px;
                }

Si solo ponemos un valor se aplicara a todos los lados. Si ponemos dos se aplicara el primero para arriba y abajo y el segundo para la derecha e izquierda. Si declaramos tres valores el primero sera de la parte superior, el segundo de los lados, y el tercero de la parte de abajo. Con los 4 se aplica con el sentido de la manecillas del reloj.

                
                .conenido{
                    margin: 0;
                    margin: 2px;
                    margin: 2px 6px;
                    margin: 2px 6px 8px;
                    margin: 3px 6px 9px 12px;
                }
